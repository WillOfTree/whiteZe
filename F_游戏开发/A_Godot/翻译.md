# GODOT翻译系统

## 备注

### 一、常用的key

```python
# 使用前缀分类
tr("ERR_")      # 错误消息
tr("TIP_")      # 提示消息
tr("UI_")       # 界面文本
tr("DIALOG_")   # 对话框
```

### 二、错误BUG

- 若没有在游戏/程序一开始的时候使用`TranslationServer.set_locale()`设置语言，当button（修改语言的Button）按下时，会有一定的概率不修改任何语言，在第二次按下才生效

## CSV设置

### 一、创建CSV文件

- **方法**：用 Excel 或 WPS 制作表格。
- **格式**：**第一列**是键（ID），**后续列**是对应语言的翻译。保存为 **UTF-8 无 BOM** 格式的 `.csv` 文件。

样式：

| key     | zh   | en      |
| :------ | :--- | :------ |
| SCORE   | 分数 | Score   |
| WELCOME | 你好 | Welcome |

### 二、编辑器中设置

1. **导入文件**：将你的 `.csv` 文件放入项目文件夹（如 `locales/`）。

    Godot 会自动识别，并生成translation文件

2. 打开 **项目 → 项目设置 → 本地化 → 翻译**，点击 **添加** 选择你的translation文件。

   - 如果无法自动识别语言，可以手动设置 `locale` 为 `zh_CN`、`en` 等。

3. **切换语言**：在你的代码中，通过一行指令即可切换语言，界面会随之改变：

## 动态数据

CSV文件

| keys          | zh_CN            | en                   |
| :------------ | :--------------- | :------------------- |
| WELCOME_LOGIN | 欢迎`{username}` | Welconme`{username}` |

gdscript：

```python
var good = tr("WELCOME_LOGIN").format({"username": username})
```

## 常用方法

``` python
# 设置语言
TranslationServer.set_locale("en")
   
# 设置为中文
TranslationServer.set_locale("zh_CN")
   
# 跟随系统语言
var user_lang = OS.get_locale_language()
TranslationServer.set_locale(user_lang)

# 获取当前设置的语言
var a = TranslationServer.get_locale()
```
