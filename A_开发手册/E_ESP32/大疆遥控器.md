

# 大疆遥控器烧录过程

## 🎯 购买清单确认

| 物品                 | 推荐型号/规格                                    | 用途                     |
| :------------------- | :----------------------------------------------- | :----------------------- |
| **ESP32 开发板**     | **ESP32-C6-DevKitC-1**（或其他 ESP32-C6 开发板） | 主控，运行遥控器程序     |
| **USB 数据线**       | USB-A 转 USB-C（支持数据传输，非充电线）         | 供电 + 烧录程序          |
| **电脑**             | Windows / Linux / macOS                          | 开发环境搭建             |
| **（可选）GPS 模块** | LC76G GNSS 模块                                  | 为拍摄的视频添加地理标签 |

## 安装与配置

- 获取大疆官方遥控器 Demo：[GitHub 地址](https://github.com/dji-sdk/Osmo-GPS-Controller-Demo.git) 
- 安装VS code + ESP-IDF

## 错误修改

> error: 'MALLOC_CAP_DEFAULT' undeclared (first use in this function)、error:'MALLOC_CAP_DMA' undeclared (first use in this function)

- 因为大疆的 Demo 代码是为旧版 ESP-IDF（如 v4.4）编写的，而你的环境是更高环境。在新版本中，内存分配相关的宏名称发生了变化
- 需要引入头文件：`#include "esp_heap_caps.h"`



## 🎮 连接大疆 Action 5 Pro

烧录成功后，开发板会自动运行遥控器程序。操作步骤如下：

| 操作             | 功能                             |
| :--------------- | :------------------------------- |
| **长按 BOOT 键** | 搜索并连接相机（LED 会闪烁蓝色） |
| **单击 BOOT 键** | 开始/停止录像                    |
| REST按钮         | 开机，重启                       |

LED 指示灯状态说明：

| LED 颜色   | 含义                      |
| :--------- | :------------------------ |
| 🔴 红色     | 系统未初始化              |
| 🟡 黄色     | BLE 已就绪，等待连接      |
| 🔵 闪烁蓝色 | 正在搜索相机              |
| 🔵 常亮蓝色 | BLE 已连接                |
| 🟢 绿色     | 协议已连接（可正常控制）  |
| 🟣 紫色     | 协议已连接 + GPS 信号正常 |
